Cheat Sheet: Tables in HTML

Creating tables for the first time can seem like a daunting prospect, with this cheat sheet you can learn the art of tables quickly and easily. By using these tags, you can create tables, for any website:

<table> Wrapper for HTML table
<thead> Defines columns for headers
<tbody> Set of rows containing table data
<tr> Table Row Container
<td> Table Row container
<tfoot> A set of table rows defining the footer

Now you have the learned the basics of tables in HTML, you can preset a wide variety of data quickly and easily

colspan Defines how many columns a td element will span
rowspan Defines how many rows a td element will span
